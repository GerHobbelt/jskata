h1. jsKata.undo

p. jsKata.undo makes undo & redo management easy.

h2. Properties

*dids* array of functions :  The actions that were done
*undids* array of functions : The actions that were undone
*canUndo()* boolean : If there are actions to undo
*canRedo()* boolean : If there are actions to redo

h2. Functions

h3. execute(doFunction, undoFunction)

*doFunction* function : The function that makes the action wanted
*undoFunction* function : The function that reverse _doFunction_

The _doFunction_ is automatically called when _execute_ is called. It enables the undo function.

h3. undo()

Undo the last action done via  the _do_ function.

h3. redo()

Redo the last action that was undone via  the _do_ function.

h3. %{color:red}deprecated% push(undoFunction)

*undoFunction* function : The function to undo

Add a undo method without adding a do method.

h2. Events

h3. onChange()

This event is raised when an action is done, undone or redone.

h3. %{color:red}deprecated% onEmpty()

This event is raised when there's nothing left to redo or undo.
